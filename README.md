# Heikin Ashi Strategy

A simple **Heikin Ashi–based trading strategy** implemented in Python using a Jupyter Notebook.

## Overview
- Uses Heikin Ashi candles to reduce market noise
- Intended for backtesting and strategy research

## Files
- `HeikinAshi_.ipynb` – Main strategy notebook

## How to Run
1. Clone the repository  
2. Open the notebook in Jupyter  
3. Run all cells
